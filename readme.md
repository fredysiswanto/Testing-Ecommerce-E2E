## Usage

- menjalankan testing

  ```sh
  npm run test

  ```

- generate report

  ```sh
  npm run test-report

  ```

- menjalankan testing api

  ```sh
  npm run test-api

  ```

- generate report test api

  ```sh
  npm run test-report-api

  ```

- menjalankan testing e2e

  ```sh
  npm run test-e2e

  ```

- generate report test e2e

  ```sh
  npm run test-report-e2e

  ```
